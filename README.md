# Event Test Site

A standalone Next.js site with 50 realistic fake events and ~300,000 participants for testing AI-powered data extraction.

## Quick Start

```bash
pnpm install
pnpm generate   # generates ~300k fake participants into src/data/events/
pnpm dev        # starts dev server at http://localhost:3000
```

## URL Structure

- `/events` — events listing (50 events, 10 per page)
- `/events/[slug]` — event overview with participant counts
- `/events/[slug]/attendees` — attendee listing (table, 20 per page)
- `/events/[slug]/speakers` — speaker listing (card grid, 20 per page)
- `/events/[slug]/exhibitors` — exhibitor directory (20 per page)
- `/events/[slug]/sponsors` — sponsor tiers (all on one page)
- `…/[id]` — individual profile pages

## Data

Generated by `pnpm generate` (script: `scripts/generate-data.ts`). Uses faker.js with seed 42 — deterministic, reproducible.

Data files are gitignored (`src/data/events/*.json`). Run `pnpm generate` to recreate them.

## Notes for AI Testing

- Section headings use varied terminology (e.g. "Registered Delegates", "Featured Voices") — never generic type labels
- ~40% of profiles have a hidden "Request Info" section revealed by clicking a button
- All pages are server-rendered with URL-based pagination — no JavaScript-only content
